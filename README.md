# peanutd

a minimal peanut server

## installation

1. clone the repo
2. run `sudo make install`

## setting up a peanut site

after installing peanutd, run `peanutd` and visit your peanut site (with a peanut client) at 127.0.0.1 on port 5050.

to learn more about the peanutmap file and how the peanut protocol works, read [peanut.md](./peanut.md)
